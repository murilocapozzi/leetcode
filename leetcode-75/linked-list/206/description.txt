# Reverse Linked List

Given the head of a singly linked list, reverse the list, and return the reversed list.

 

Example 1:
Image used in example: https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]


Example 2:
Image used in example: https://assets.leetcode.com/uploads/2021/02/19/rev1ex2.jpg
